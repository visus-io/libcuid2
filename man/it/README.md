# Pagine di manuale libcuid2 - Italiano

Questa directory contiene le traduzioni italiane delle pagine di manuale di libcuid2.

## Stato della traduzione

- [ ] cuid2gen.1 - Manuale dello strumento CLI
- [ ] libcuid2.3 - Manuale di riferimento API
- [ ] libcuid2.7 - Manuale di panoramica e architettura

## Come contribuire

Vedere `../LOCALIZATION.md` per le linee guida di traduzione.

## Utilizzo

Dopo l'installazione:
```bash
LANG=it_IT.UTF-8 man cuid2gen
LANG=it_IT.UTF-8 man 3 libcuid2
LANG=it_IT.UTF-8 man 7 libcuid2
```

Oppure prima dell'installazione:
```bash
man -l man/it/man1/cuid2gen.1
man -l man/it/man3/libcuid2.3
man -l man/it/man7/libcuid2.7
```
