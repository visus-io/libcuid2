# Справочные страницы libcuid2 - Русский

Этот каталог содержит русские переводы справочных страниц libcuid2.

## Статус перевода

- [ ] cuid2gen.1 - Справочная страница инструмента CLI
- [ ] libcuid2.3 - Справочная страница API
- [ ] libcuid2.7 - Справочная страница обзора и архитектуры

## Как внести вклад

См. `../LOCALIZATION.md` для руководства по переводу.

## Использование

После установки:
```bash
LANG=ru_RU.UTF-8 man cuid2gen
LANG=ru_RU.UTF-8 man 3 libcuid2
LANG=ru_RU.UTF-8 man 7 libcuid2
```

Или до установки:
```bash
man -l man/ru/man1/cuid2gen.1
man -l man/ru/man3/libcuid2.3
man -l man/ru/man7/libcuid2.7
```
