# libcuid2 매뉴얼 페이지 - 한국어

이 디렉토리에는 libcuid2 매뉴얼 페이지의 한국어 번역이 포함되어 있습니다.

## 번역 상태

- [ ] cuid2gen.1 - CLI 도구 매뉴얼
- [ ] libcuid2.3 - API 참조 매뉴얼
- [ ] libcuid2.7 - 개요 및 아키텍처 매뉴얼

## 기여하기

번역 지침은 `../LOCALIZATION.md`를 참조하세요.

## 사용법

설치 후:
```bash
LANG=ko_KR.UTF-8 man cuid2gen
LANG=ko_KR.UTF-8 man 3 libcuid2
LANG=ko_KR.UTF-8 man 7 libcuid2
```

또는 설치 전:
```bash
man -l man/ko/man1/cuid2gen.1
man -l man/ko/man3/libcuid2.3
man -l man/ko/man7/libcuid2.7
```
