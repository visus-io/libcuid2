# Strony podręcznika libcuid2 - Polski

Ten katalog zawiera polskie tłumaczenia stron podręcznika libcuid2.

## Status tłumaczenia

- [ ] cuid2gen.1 - Podręcznik narzędzia CLI
- [ ] libcuid2.3 - Podręcznik referencyjny API
- [ ] libcuid2.7 - Podręcznik przeglądu i architektury

## Jak wnieść wkład

Zobacz `../LOCALIZATION.md` dla wytycznych dotyczących tłumaczenia.

## Użycie

Po instalacji:
```bash
LANG=pl_PL.UTF-8 man cuid2gen
LANG=pl_PL.UTF-8 man 3 libcuid2
LANG=pl_PL.UTF-8 man 7 libcuid2
```

Lub przed instalacją:
```bash
man -l man/pl/man1/cuid2gen.1
man -l man/pl/man3/libcuid2.3
man -l man/pl/man7/libcuid2.7
```
