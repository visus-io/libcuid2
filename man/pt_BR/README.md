# Páginas de manual do libcuid2 - Português (Brasil)

Este diretório contém as traduções para português brasileiro das páginas de manual do libcuid2.

## Status da tradução

- [ ] cuid2gen.1 - Manual da ferramenta CLI
- [ ] libcuid2.3 - Manual de referência da API
- [ ] libcuid2.7 - Manual de visão geral e arquitetura

## Como contribuir

Consulte `../LOCALIZATION.md` para as diretrizes de tradução.

## Uso

Após a instalação:
```bash
LANG=pt_BR.UTF-8 man cuid2gen
LANG=pt_BR.UTF-8 man 3 libcuid2
LANG=pt_BR.UTF-8 man 7 libcuid2
```

Ou antes da instalação:
```bash
man -l man/pt_BR/man1/cuid2gen.1
man -l man/pt_BR/man3/libcuid2.3
man -l man/pt_BR/man7/libcuid2.7
```
