# Pages de manuel libcuid2 - Français

Ce répertoire contient les traductions françaises des pages de manuel de libcuid2.

## État de la traduction

- [ ] cuid2gen.1 - Manuel de l'outil CLI
- [ ] libcuid2.3 - Manuel de référence de l'API
- [ ] libcuid2.7 - Manuel de présentation et d'architecture

## Contribuer

Voir `../LOCALIZATION.md` pour les directives de traduction.

## Utilisation

Après l'installation :
```bash
LANG=fr_FR.UTF-8 man cuid2gen
LANG=fr_FR.UTF-8 man 3 libcuid2
LANG=fr_FR.UTF-8 man 7 libcuid2
```

Ou avant l'installation :
```bash
man -l man/fr/man1/cuid2gen.1
man -l man/fr/man3/libcuid2.3
man -l man/fr/man7/libcuid2.7
```
