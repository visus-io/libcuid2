# Páginas de manual do libcuid2 - Português (Europeu)

Esta directoria contém as traduções para português europeu das páginas de manual do libcuid2.

## Estado da tradução

- [ ] cuid2gen.1 - Manual da ferramenta CLI
- [ ] libcuid2.3 - Manual de referência da API
- [ ] libcuid2.7 - Manual de visão geral e arquitectura

## Como contribuir

Consulte `../LOCALIZATION.md` para as directrizes de tradução.

## Utilização

Após a instalação:
```bash
LANG=pt_PT.UTF-8 man cuid2gen
LANG=pt_PT.UTF-8 man 3 libcuid2
LANG=pt_PT.UTF-8 man 7 libcuid2
```

Ou antes da instalação:
```bash
man -l man/pt/man1/cuid2gen.1
man -l man/pt/man3/libcuid2.3
man -l man/pt/man7/libcuid2.7
```
