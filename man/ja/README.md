# libcuid2 マニュアルページ - 日本語

このディレクトリには、libcuid2 マニュアルページの日本語訳が含まれています。

## 翻訳状況

- [ ] cuid2gen.1 - CLI ツールマニュアル
- [ ] libcuid2.3 - API リファレンスマニュアル
- [ ] libcuid2.7 - 概要とアーキテクチャマニュアル

## 貢献

翻訳ガイドラインについては `../LOCALIZATION.md` を参照してください。

## 使用方法

インストール後:
```bash
LANG=ja_JP.UTF-8 man cuid2gen
LANG=ja_JP.UTF-8 man 3 libcuid2
LANG=ja_JP.UTF-8 man 7 libcuid2
```

またはインストール前:
```bash
man -l man/ja/man1/cuid2gen.1
man -l man/ja/man3/libcuid2.3
man -l man/ja/man7/libcuid2.7
```
