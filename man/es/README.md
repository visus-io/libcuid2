# Páginas de manual de libcuid2 - Español

Este directorio contiene las traducciones al español de las páginas de manual de libcuid2.

## Estado de traducción

- [ ] cuid2gen.1 - Manual de la herramienta CLI
- [ ] libcuid2.3 - Manual de referencia de la API
- [ ] libcuid2.7 - Manual de descripción general y arquitectura

## Contribuir

Consulte `../LOCALIZATION.md` para las directrices de traducción.

## Uso

Después de la instalación:
```bash
LANG=es_ES.UTF-8 man cuid2gen
LANG=es_ES.UTF-8 man 3 libcuid2
LANG=es_ES.UTF-8 man 7 libcuid2
```

O antes de la instalación:
```bash
man -l man/es/man1/cuid2gen.1
man -l man/es/man3/libcuid2.3
man -l man/es/man7/libcuid2.7
```
