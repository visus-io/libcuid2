## 使用方法

安装后:
```bash
LANG=zh_CN.UTF-8 man cuid2gen
LANG=zh_CN.UTF-8 man 3 libcuid2
LANG=zh_CN.UTF-8 man 7 libcuid2
```

或安装前:
```bash
man -l man/zh_CN/man1/cuid2gen.1
man -l man/zh_CN/man3/libcuid2.3
man -l man/zh_CN/man7/libcuid2.7
```
