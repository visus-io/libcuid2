## Beitragen

## Verwendung

Nach der Installation:
```bash
LANG=de_DE.UTF-8 man cuid2gen
LANG=de_DE.UTF-8 man 3 libcuid2
LANG=de_DE.UTF-8 man 7 libcuid2
```

Oder vor der Installation:
```bash
man -l man/de/man1/cuid2gen.1
man -l man/de/man3/libcuid2.3
man -l man/de/man7/libcuid2.7
```
